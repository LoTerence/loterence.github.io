Altcoin Charter

A RESTful CRUD web application where users are able to view a chart displaying the price histories of a selected cryptocurrency. Authenticated users are able to save a personal watchlist onto their account.

Built with the MERN Stack: NodeJS, ExpressJS, ReactJS + ReduxJS, and MongoDB. Data visualization was built with the Uber react-vis library. Mongoose was used for the database connection, which is hosted on mLab. The app itself is deployed on Heroku. Passport was used for user authentication. The cryptocoin data came from the cryptocompare.com API.

Live version is available at: https://altcoin-charter.herokuapp.com/

Github repo link: https://github.com/LoTerence/altcoin-charter

Urbandict Bot

A groupme chat bot that allows users to search for the urban dictionary definition of a word within the chat room. Type in /urban followed by a word you want to look up and the bot will respond with its definition. Comes in really handy when your friends use new slang that no one in the chat understands!

This bot was built with NodeJS using an ExpressJS server and is deployed on Heroku. It makes API requests to urbandictionary.com for its definitions.

Github repo link: https://github.com/LoTerence/urbandict-gmebot
